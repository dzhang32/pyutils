pyutils.here module¶

pyutils.here.here(script_path: Optional[str] = None, *args: str) → str¶

Obtain the relative path to directory of the executed script.

Useful for making projects self-contained, for example by making results path relative to the script where they are generated.

Args:

script_path (str): path to the current executed script. args (str): relative paths to the file or directory of interest.

Returns:

str: Absolute path to file/directory of interest.

pyutils.timer module¶

pyutils.timer.timer(num_times: int)¶

Decorator function that times the inputted function.

Runs the inputted function num_times, then prints and returns average time taken

Args:

num_times (int): Number of times to run the function.

Returns:

float: Average time taken to run function across num_times iterations.
